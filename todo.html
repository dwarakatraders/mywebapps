To-Do Management System

This To-Do List Management System is a comprehensive tool for managing tasks with robust features for adding, displaying, filtering, sorting, and performing actions on tasks. The user interface is designed to be responsive and user-friendly, ensuring an efficient workflow for managing tasks.

Features

Add Task:

Users can add new tasks with a task name and assignee. Tasks are assigned a unique ID and the current status is set to "Open". Task creation date is automatically recorded.

Task Display:

Tasks are displayed in a paginated list, showing task name, status, assignee, creation date, and completion date (if applicable). Task details can be toggled to show or hide additional information.

Task Actions:

Mark as Completed: Users can mark tasks as completed using a link. Re-Open Task: Completed tasks can be reopened. Delete Task: Tasks can be deleted.

Filter and Sort:

Tasks can be filtered by status and assignee. Tasks can be sorted by creation date or task name.

Pagination:

Tasks are displayed in pages with navigation controls to move between pages. Items per page is set to 3.

Responsive Design:

Filter and sort options are displayed in a single row on larger screens. On mobile devices, filter options display one by one. Task list layout adjusts for different screen sizes.
